Bubble sort Algorithm

Ascending order

  1. Define a function for sorting with one parameter
  2. Get the length of the array used as limitation
  3. Compare 2 element from the beginning of the array
  4. Whenever the second element is greater than the first, swap them
  5. Compare the next pair of elements and repeat step 4
  6. Repeat step 3 to 5 until the array is sorted

End
